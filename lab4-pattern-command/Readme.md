Разработать модуль электронной библиотечной системы, в котором операции обслуживания книг реализованы с помощью паттерна Команда. Система должна поддерживать действия: выдача книги читателю, возврат книги, продление срока выдачи, бронирование книги и отмена бронирования.

Использование паттерна Команда целесообразно, поскольку библиотечная система содержит набор независимых операций, которые удобно представлять в виде отдельных объектов-команд.

Параметры - книга, читатель, дата
